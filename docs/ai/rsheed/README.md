# Rsheed

Persoenlicher Arbeitsbereich fuer optionale KI-Artefakte.

Empfohlenes Muster pro Issue:

- `issue-<nummer>-seed.md` fuer Ziel, offene Fragen, Fortschritt und naechste Schritte
- weitere Dateien nur bei Bedarf, z. B. fuer einen groesseren Plan oder eine Architekturentscheidung

Beispiel:

- `issue-60-seed.md`
- `issue-30-seed.md`
